Commands
vim filename = opens a new or exisiting file for editing
Esc = go to Normal Mode
i = (insert) start editing
:q = (quit) exit the document, but will not let you exit if you have unsaved changes
:q! = (quit, forcefully) will exit the document even if unsaved changes
:w = (write) save the document
:wq = (write, quit) save and exit
