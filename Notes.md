## Building Access
>Visit the following website:  
https://spacek.korea.ac.kr/user/userMain.do  

>login with KUPID credenials  

>Go to the menu option: 출입신청  

>Choose the following options for the PALM lab rooms:  
건물명 (Building Name):  우정정보통신관  
층명 (Floor): 지상4층  
호실명 (Room Name): [407A]프로그램 분석, 학습, 마이닝 그룹  
신청 기간 (Application Period): Select appropriate start and end date  
신분증 선택 (ID types): Select both the options => 카드 신분증 (Physical Card) and 체크박스 모바일 신분증 (Mobile Id)  

>Click on the button 호실 추가 which will add the access entry to the table

>Fill the fields in the form labeled: 신청 정보  
신청 제목 (Application Title): Give a suitable title => "Application for Lab Access"  
신청 내용 (Application Details): Give a suitable explanation => "Applying for access to lab"  

>Once done click on 출입 신청
